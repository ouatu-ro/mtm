# Concepts

MTM keeps the short vocabulary used by the CLI in
[`concepts.toml`](concepts.toml).

Use the CLI to print the same vocabulary:

```sh
uv run mtm concepts
uv run mtm concepts UTMBandArtifact
```

`mtm inspect` also points to the relevant concept for each artifact it
recognizes.
